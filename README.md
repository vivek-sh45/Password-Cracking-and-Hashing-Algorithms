# Password Cracking and Hashing Algorithms

## Internship Details
Company Name: Codec Technologies India  
Domain: Cyber Security  
Project Type: Internship Training Project  

## Objective
Analyze and implement brute-force password cracking tools while understanding hash functions and salting techniques.

## Tools & Technologies
- Python  
- Hashcat  
- John the Ripper  
- Kali Linux  

## Project Features
- Password hashing using MD5, SHA1, SHA256  
- Password verification system  
- Brute-force password cracking using wordlist  
- Demonstration of cryptographic weaknesses  

## Files Description
- hashing.py → Generates password hashes  
- verify.py → Verifies password against hash  
- bruteforce.py → Performs brute-force cracking  
- crack.py → Main cracking script  
- wordlist.txt → Password dictionary  

## Skills Learned
- Hashing algorithms  
- Brute force attack concepts  
- Password cracking methodology  
- Cryptographic security basics
